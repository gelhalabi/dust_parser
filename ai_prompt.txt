Dust Sensor Monitoring Project - Session Starting Point

PROJECT OVERVIEW
This is a Python-based verification tool for monitoring two dust sensors located at 2m and 24m heights. The tool connects to sensors at 10.40.200.32:50003, reads real-time particle measurement data, and displays formatted results.

CURRENT IMPLEMENTATION STATUS
- Working Python async implementation
- Successfully parsing CSV-style data format: date,time,MT1[8 values],MT2[8 values]
- Real-time data display with averages
- Configuration via YAML
- Basic error handling and logging

TECHNICAL SPECIFICATIONS
1. Network Configuration:
   - Host: 10.40.200.32
   - Port: 50003
   - Protocol: TCP/Telnet
   - Timeout: 10 seconds
   - Read interval: 1.0 seconds

2. Data Format:
   - CSV-style format
   - Example: "25/05/07,22:46:35,MT1 191,MT1 21,...,MT2 150,MT2 20,..."
   - 18 fields total (date, time, 8 MT1 values, 8 MT2 values)

3. Key Components:
   - SensorClient: Handles network communication
   - DataParser: Processes raw sensor data
   - YAML configuration
   - Async main loop
